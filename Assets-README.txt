A pasta Assets foi configurada para ser ignorada nos commits para o git (veja o arquivo .gitignore). Isso evita que o tamanho do repositório cresça demais ao longo de um projeto.

Baixe o arquivo Assets.zip:
https://drive.google.com/file/d/1tb8g0XdnE341Zm43A08-7vlyETl0HFwZ/view?usp=drive_link

Será necessário descompactar o arquivo (mas eu estou na dúvida sobre como isso deve ser feito)
a) se os Assets aparecem no projeto da Godot e são marcados como arquivos inexistentes, descompacte o conteúdo dos Assets direto na pasta do projeto.
b) se os Assets não aparecem no projeto da Godot, descompacte o conteúdo dos Assets em uma pasta que não seja a pasta do projeto e depois arraste a pasta Assets do seu Explorador de Arquivos para a aba "Arquivos" da Godot.
